# HotSauce and Ketchup

A Pizza store that helps customize topping and order online

## Tech stacks:

- C#
- ASP.NET Core (Razor Pages)
- Entity Framework
- SQL Server
- Sass

## System Requirements:

- .NET 5.0
- SQL Server 2019 or above
- Visual Studio 2022
